# Learning Objectives and Key Topics Template

## Learning Objectives Format

Each chapter should have a "Learning Objectives" section that follows this format:

```markdown
## Learning Objectives

- Understand [key concept 1]
- Learn [key concept 2]
- Explore [key concept 3]
- Apply [key concept 4] (if applicable)
```

## Key Topics Format

Each chapter should have a "Key Topics" section that follows this format:

```markdown
## Key Topics

1. [Topic 1]
2. [Topic 2]
3. [Topic 3]
4. [Topic 4] (if applicable)
```

## Guidelines

### Learning Objectives
- Use action verbs: "Understand", "Learn", "Explore", "Apply", "Identify", "Describe", "Explain"
- Focus on what the student will be able to do after completing the chapter
- Keep objectives specific and measurable
- Limit to 3-5 main objectives

### Key Topics
- Use numbered lists (ordered) to indicate the sequence of topics
- Keep topic descriptions concise but descriptive
- Align topics with learning objectives
- Use consistent terminology across chapters
